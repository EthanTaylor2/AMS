# MKDIR

This part of code is used for mkde directories which will save optical flow data.

----
## How to run the code
1. unzip filename.rar
2. input:
```
   g++ -o mkdtest.cpp mkdtest
   g++ -o mkdtrain.cpp mkdtrain
   g++ -o mkdvalid.cpp mkdvalid
   ./mkdtest
   ./mkdtrain
   ./mkdvalid
   ```
