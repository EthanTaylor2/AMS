# MKDIR

This part of code is used for mkde directories which will save optical flow data.

----
## How to run the code
1. unzip filename.rar
2. input:
```
   g++ -o mkdtest mkdtest.cpp
   g++ -o mkdtrain mkdtrain.cpp 
   g++ -o mkdvalid mkdvalid.cpp
   ./mkdtest
   ./mkdtrain
   ./mkdvalid
   ```
